# AI Image Generator

This project is a simple AI Image Generator built using Python and Stable Diffusion in Google Colab. The model generates images from text prompts using Generative AI.

## Technologies Used
- Python
- Google Colab
- PyTorch
- Diffusers
- Stable Diffusion

## Features
- Text-to-image generation
- AI-generated realistic images
- Easy and beginner-friendly implementation

## Sample Prompt
"A futuristic city at night"

## Output
The model generates an image based on the given text prompt.
